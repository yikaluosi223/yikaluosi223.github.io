这个笔记介绍如何浏览和使用这个数字花园。

## 页面类型

### 📝 博客文章

位于 `[[posts/|博客]]` 区域，是按时间排列的长文章。适合深度阅读。

### 🌿 知识笔记

位于 `[[notes/|笔记]]` 区域，是按照 Zettelkasten 方法组织的原子化笔记。每个笔记聚焦一个概念或知识点，通过 `[[wikilinks]]` 相互连接形成知识网络。

笔记有三种成熟度：

- 🌱 种子 (Seedling)：刚创建，内容还在完善
- 🌿 萌芽 (Budding)：内容基本稳定，链接逐步建立
- 🌲 常青 (Evergreen)：经过多次打磨，结构清晰

### 🚀 项目展示

位于 `[[projects/|项目]]` 区域，记录项目经验和成果。

## 如何使用

### 浏览

- 通过顶部菜单导航不同区域
- 使用 [[tags/|标签]] 浏览相关主题
- 使用 [[search/|搜索]] 查找特定内容
- 点击文章中的链接在不同页面间跳转

### 阅读笔记技巧

笔记之间有大量双向链接。当你阅读一篇笔记时：

1. 先看标题和第一段，了解主题
2. 阅读正文内容
3. 查看底部 "链接到此页面" 的部分，发现相关笔记
4. 跟随感兴趣的链接继续探索

> [!tip] 提示
> 这个网站就像一座花园，每条小路（链接）都可以通向不同的风景。不要急着看完所有内容，慢慢探索就好。
