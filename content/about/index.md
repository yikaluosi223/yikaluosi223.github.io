## 关于这个博客

这个博客使用 [Hugo](https://gohugo.io/) 构建，主题基于 [PaperMod](https://github.com/adityatelange/hugo-PaperMod)，并针对 Obsidian 进行了深度定制。

### 技术栈

- **静态网站生成**: Hugo
- **内容编辑**: Obsidian
- **部署**: GitHub Pages
- **版本管理**: Git

### 特色

- Wikilink 双向链接支持
- Obsidian 风格 Callout 渲染
- 反向链接显示
- 知识图谱可视化
- 中文排版优化
- 暗色模式
